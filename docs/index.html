Rock-Paper-Scissors API Demo

API Usage Instructions

This API allows you to play Rock-Paper-Scissors via REST and WebSocket endpoints.

  • POST /join: Join the game with a player name. Example: {"id": "", "name": "Alice"}
  • GET /players: List all joined players.
  • GET /random-move: Get a random move (ROCK, PAPER, SCISSORS).
  • WebSocket /game?playerId={id}: Play the game by sending moves: {"playerId": "your-id", "move": "ROCK"}
